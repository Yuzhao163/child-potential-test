# 儿童潜能评估平台

发现孩子的无限可能 - 基于多元智能理论的儿童潜能评估系统

## 功能特点

### 12大评估维度
- **学术智能** - 语言表达、数学逻辑、阅读理解等
- **艺术创造** - 视觉艺术、创意设计、想象力等
- **体育运动** - 大运动、精细动作、协调性等
- **科技素养** - 编程思维、逻辑推理、数字素养等
- **社交情感** - 沟通能力、团队协作、同理心等
- **音乐韵律** - 节奏感知、音高感、音乐记忆等
- **语言发展** - 词汇量、语法结构、表达能力等
- **逻辑思维** - 模式识别、因果理解、批判思维等
- **领导力** - 组织能力、决策能力、责任感等
- **自律专注** - 专注时长、冲动控制、时间管理
- **想象力** - 幻想力、好奇心、创新思维等
- **生活自理** - 自我照顾、日常规划、安全意识等

### 核心功能
- **多元评估**：支持同时选择3-5个类别进行综合评估
- **智能问卷**：每个维度4道题，共5级评分
- **实时进度**：显示答题进度和用时
- **自动保存**：支持中断后继续评估
- **丰富图表**：雷达图、柱状图、热力图等多维可视化
- **个性化建议**：基于评估结果生成发展建议
- **历史记录**：保存历次评估结果对比
- **数据导出**：支持PDF打印和JSON导出
- **深色模式**：支持浅色/深色主题切换

## 技术栈

- **前端**：纯HTML5 + CSS3 + JavaScript (ES6+)
- **图表**：Chart.js 4.x (CDN)
- **字体**：Noto Sans SC (Google Fonts CDN)
- **存储**：LocalStorage (本地数据持久化)
- **路由**：Hash Router (SPA单页应用)

## 快速开始

1. 直接在浏览器中打开 `index.html` 文件
2. 或使用本地服务器：
   ```bash
   # Python 3
   python -m http.server 8080

   # Node.js
   npx serve .
   ```
3. 访问 `http://localhost:8080`

## 页面说明

| 路由 | 描述 |
|------|------|
| `#home` | 首页，展示12大评估类别 |
| `#assessment` | 评估设置，输入信息并选择类别 |
| `#test` | 在线测评，答题界面 |
| `#result` | 评估报告，图表和建议 |
| `#history` | 历史记录，查看过往评估 |
| `#about` | 关于系统 |

## 数据存储

所有数据均存储在浏览器本地LocalStorage中：
- `assessment_progress` - 未完成的评估进度
- `assessment_history` - 历史评估记录（最多10条）
- `theme` - 主题偏好设置

## 隐私说明

- 数据完全存储在本地，不会上传到任何服务器
- 清除浏览器数据将删除所有评估记录
- 可随时手动删除历史记录

## 浏览器兼容

- Firefox 75+
